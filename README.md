Sharing app server-side application
============================

## Maven
### Running locally

`mvn appengine:run`

To use vist: http://localhost:8080/

### Deploying

`mvn appengine:deploy`

To use vist:  https://YOUR-PROJECT-ID.appspot.com

### Browsing local datastore
http://localhost:8080/_ah/admin

## Testing

`mvn verify`
